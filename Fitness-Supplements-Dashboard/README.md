# Fitness & Supplements Company Dashboard

## Project Overview
This project analyzes data for a fitness & supplements company to track business performance and support data-driven decisions.

## Business Questions
- How are revenue and profit performing over time?
- Which regions generate the highest sales?
- What product categories drive the most revenue?
- How do customers and orders vary by region?

## Key KPIs
- Revenue
- Return Value
- Profit
- Orders
- Return Orders
- Quantity Sold
- Return Quantity
- Customers
- Return Customers
- Customers Without Orders
- Cost
- Average Order Value (AOV)
- Year-over-Year (YoY) Variance

## Insights Highlights
- Revenue growth is driven mainly by higher order volume rather than price increases.
- Profit margins vary significantly by region due to differences in costs and return rates.
- High-revenue product categories also experience higher return quantities.
- A segment of customers places no orders, indicating re-engagement opportunities.
- YoY analysis shows overall business growth with seasonal fluctuations.
